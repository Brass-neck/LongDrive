# 跳转结构 和 紧密结构

- 紧密结构
  - 数组，连续一块内存，如果长度太大，有可能是多个连续片段连起来的，每个片段的尾部指向下一个片段
- 跳转结构
  - 链表
  - 双向链表
  - 图，多个方向的链表
  - list

# 数据结构设计

- 场景：一个数组，总是需要求 L ~ R 上数字的和
  - 设计一个一维数组，前缀和
  - arr = [0,1,2,34,5]， helpArr = [0,1,3,37,42]，helpArr[i] 就是 0~i 的和
  - 如果要计算 2~4 的累加和，helpArr[4] - helpArr[1]

```js
// 生成一个 help 数组
let arr = [0, 1, 2, 3, 4, 5]

let help = []
help[0] = arr[0]

for (let i = 1; i < N; i++) {
  help[i] = help[i - 1] + arr[i]
}
```
